# solar-system-web-page
Site feito em HTML e CSS destinado a manter informações sobre a história do sistema solar.

## Descrição

Página web para acesso a informações sobre o início e desenvolvimento do sistema solar. Inclui protótipo de sistema baseado em login e registro.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript (futuramente para implementação de validação de login e de registro)

## Funcionalidades

- Protótipo de login
- Protótipo de registro
- Homepage com informações sobre a história do sistema solar
- Botões para redirecionamento para páginas de login e de registro

## Disciplina

Projeto final feito para comprovação dos conhecimentos técnicos adquiridos durante disciplina opcional de desenvolvimento web em 2026.

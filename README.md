# web-page
Local web page for accessing a home page screen via the login and register system.

## Descrição

Página web para acesso a informações sobre o início e desenvolvimento do sistema solar. Inclui protótipo de sistema baseado em login e registro.

## Tecnologias utilizadas

- HTML
- CSS

## Funcionalidades

- Protótipo de login
- Protótipo de registro
- Homepage com informações sobre a história do sistema solar
- Botões para redirecionamento para páginas de login e de registro

## Disciplina

Projeto final feito durante disciplina de desenvolvimento web
